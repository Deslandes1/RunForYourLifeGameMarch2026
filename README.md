# RunForYourLifeGameMarch2026 – Escape the Crossfire

A tense, fast‑paced 2D survival game set in a Haitian village under crossfire.  
You play as a father leading his three sons to safety while avoiding bullets from bandits and police.

---

## 🎮 How to Play

- **Arrow keys** – move the father (the three sons follow smoothly).
- **Hold SHIFT** – run faster to dodge bullets.
- **Goal** – get the whole family into the **green Safe Haven** area on the right side of the screen.
- **One hit** from any bullet ends the game.

The game includes an immersive ambient soundtrack with random gunshot sounds (requires you to click the start button to enable audio).

---

## 🌐 Play Online (No Installation Required)

The game is hosted on **Streamlit Cloud** and can be played directly in any modern web browser.

### On a Computer (Windows / Mac / Linux)

1. Open your web browser (Chrome, Firefox, Edge, Safari).
2. Go to the game URL: **[INSERT YOUR STREAMLIT APP URL HERE]**  
   (e.g., `https://runforyourlifegamemarch2026.streamlit.app`)
3. Click the **KÒMANSE CHAPOUI** button to start.
4. Use the arrow keys and SHIFT key to control the father.
5. Escape the crossfire and reach the safe zone!

> 💡 **Tip:** For the best experience, use a desktop or laptop with a physical keyboard.

### On a Phone (iOS / Android)

The game is designed for keyboard controls, but you can still play on a phone with a little setup:

- **Option 1 – Bluetooth keyboard:** Connect a Bluetooth keyboard to your phone, then open the game URL in Chrome or Safari. Use the arrow keys and SHIFT as usual.
- **Option 2 – On‑screen keyboard:**  
  1. Open the game URL in your browser.  
  2. Tap the address bar and select **“Request Desktop Site”** (in Chrome) or **“Desktop Website”** (in Safari).  
  3. Tap the game canvas to show the on‑screen keyboard (it may appear automatically).  
  4. Use the arrow keys on the keyboard that appears.  
  5. Hold the **SHIFT** key on the on‑screen keyboard to run (tap and hold).

> ⚠️ Note: The game is best enjoyed on a computer. On a phone, the experience may be less comfortable due to screen size and keyboard limitations.
